# mi-spectral

A finite-M spectral correction for likelihood-based hypothesis tests under congenial multiple imputation.

Working manuscript targeting [JAIGP](https://jaigp.org). Positions as the spectral completion of the Meng–Rubin (1992) → Chan–Meng (2022) lineage: drops the EFMI (Equal Fraction of Missing Information) assumption by replacing scalar null calibration with the full eigenvalue spectrum of RIV_⊥.

See `CLAUDE.md` for scope and `IDEAS.md` for the directional anchor.
